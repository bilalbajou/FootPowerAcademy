## Setup

1. **Clone the repo:**
    ```sh
    git clone https://github.com/bilalbajou/FootPowerAcademy.git
    cd FootPowerAcademy
    ```
2. **Install dependencies:**
    ```sh
    composer install
    npm install
    npm run dev
    ```
3. **Copy `.env.example` to `.env` and set your environment variables.**
4. **Run migrations:**
    ```sh
    php artisan migrate
    ```
5. **Start the server:**
    ```sh
    php artisan serve
    ```
